# Contributing to Mulima Finance

Thank you for considering contributing to the Mulima Finance project! We appreciate your interest and contributions.

## Contribution Guidelines
- Please ensure your contributions align with the project's goals and values.
- Discuss new features or significant changes with maintainers before starting work on them.

## Development Setup
1. **Clone the Repo**: `git clone https://github.com/CodeEdge-ai/mulima-finance.git`
2. **Install Dependencies**: Run `npm install` or `yarn install` in the root directory.
3. **Run the Application**: Use `npm start` or `yarn start` to start the development server.

## Coding Standards
- Follow the [JavaScript Standard Style](https://standardjs.com/) for styling your code.
- Write descriptive commit messages that explain the changes you made.

## Pull Request Process
1. Ensure your code changes are well tested.
2. Open a pull request from your feature branch to the `main` branch.
3. Provide a clear and concise description of your changes.
4. Respond to any review comments appropriately.

## Issue Guidelines
- Before submitting a new issue, please check if your issue already exists.
- Provide enough detail for others to reproduce any bugs you encounter.
- Label issues appropriately to help us manage them more effectively.

Thanks for contributing! We look forward to your input and feedback!
